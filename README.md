# memory_disposal
メモリー在庫の廃棄・供給調整を管理する Flask アプリです。

## 機能
- 廃棄対象在庫の登録（アイテム名 / 数量 / 設備年数）
- 廃棄方法の選択
  - 物理破壊
  - リサイクル
  - 生産削減
  - 援助転用
- コスト計算・期待効果計算・純効果（期待効果 - コスト）の可視化
- 環境影響評価（低/中/高）とリスク評価（低/中/高）
- 推奨廃棄方法の自動判定
  - 20年以上設備: 生産削減
  - 大量在庫: リサイクル / 援助転用を優先
- 一括自動計画（未設定アイテムへ推奨方法を一括適用）
- 承認ワークフロー（未承認 -> 承認済 -> 廃棄済）

## セットアップ
```bash
pip install -r requirements.txt
python app.py
```

ブラウザで `http://127.0.0.1:5000` を開いてください。
